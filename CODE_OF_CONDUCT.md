# Code of Conduct

## Our Pledge

We want InPlace to be a respectful, constructive, and collaborative project.
Contributors, maintainers, and participants are expected to treat each other with professionalism and respect.

## Expected Behavior

- Be respectful in communication
- Give feedback on code and ideas, not on people
- Assume good intent, but be clear and direct
- Help keep discussions focused and productive

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language
- Personal attacks
- Deliberate disruption of discussions or reviews
- Publishing private information without permission

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and review interactions.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.

## Reporting

If you encounter behavior that violates this code of conduct, please report it privately to the maintainers through the security or contact channel defined for the repository.
